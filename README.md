🎬 StreamCore
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
A modern, high-performance media streaming client with automatic updates, TV support, and a premium UI experience.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Overview
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

StreamCore is a sleek, responsive media streaming application built with Flutter.
Designed for both mobile and TV environments, it delivers a polished, powered by my own media backend.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Built for:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
📱 Android
📺 Android TV

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
🚀 Features
🎥 Premium UI
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Left-side navigation layout
Hero carousel with auto-slide animation
Animated dots indicator
Subtle SC glow overlay
Scroll-based hero shrink animation
Smooth poster fade-in loading

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
📺 TV Optimised
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
D-pad navigation support
Focus animations (scale + glow)
Larger hit targets
Smooth grid navigation

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
📂 Media Browsing
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Movies & TV Shows sections
Genre filtering
Default sort: Newest
Progress bar
Watched indicator (✅ when ≥ 90%)
Continue Watching with removable items

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
🔄 Automatic Updates
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Silent update check on app launch
Manual Check for updates button
GitHub Releases integration
Android APK download + installer

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
📦 Installation (Android)
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Download the latest APK from the Releases
Enable Install unknown apps for StreamCore.
Install the APK.
Open the app — updates will be checked automatically.
